# AGENTS.md

## Project Structure
- React + Vite portfolio site
- Entry point: src/main.jsx → renders App.jsx
- Sections: Hero, About, Skills, Projects, Contact

## Key Commands
- `npm run dev` - Start dev server
- `npm run build` - Create production build
- `npm run lint` - Run ESLint

## Code Structure
- `src/components/sections/` - Page sections (Hero, About, Skills, Projects, Contact)
- `src/components/ui/` - Reusable components (SkillCard, ProjectCard, ProjectCarousel)
- `src/components/layout/` - Header, Footer
- `src/data/` - skills.js, projects.json

## Tech Stack
- Vite + React 19
- Tailwind CSS v4
- Framer Motion (animations)
- React Scroll (smooth navigation)
- Lucide React (icons)
